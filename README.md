# 📝 BlogFlow – Where Thoughts Find Their Rhythm

> **"Your voice is your brand. BlogFlow is just the microphone."**

BlogFlow is a full-stack blogging platform built to provide a clean, distraction-free space where ideas take center stage. Whether you're sharing tutorials, documenting your journey, or expressing your thoughts, BlogFlow offers a smooth and elegant writing experience.

---

## 🌟 About the Project

BlogFlow is designed with a **minimalist-first** philosophy, allowing users to focus on writing instead of navigating unnecessary complexity. It combines a modern React frontend with a powerful Node.js backend to create a seamless blogging experience.

---

## ✨ Features

- 🔐 Secure user authentication
- ✍️ Create, edit, and delete blog posts
- 💬 Community interaction through comments
- 📱 Fully responsive user interface
- 🎨 Clean, cinematic, and minimalist design
- ⚡ Fast frontend deployment with Netlify
- ☁️ Persistent backend powered by Node.js and MongoDB

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcrypt.js

### Deployment
- **Frontend:** Netlify
- **Backend:** Render
- **Database:** MongoDB Atlas
